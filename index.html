Setup Guide

Follow these steps to add RiftCord to your server

1

Invite the Bot

Add RiftCord to your Discord server with the necessary permissions.

Add to Server
2

Configure Settings

Use !setup to configure the bot for your server and set up required channels.

!setup #general #bot-commands
3

Set Up Riot API (Optional)

For full LoL features, add your Riot API key (admin only).

!setapikey YOUR_RIOT_API_KEY

Get your API key from the Riot Developer Portal

4

Customize & Enjoy

Customize moderation settings, enable/disable features, and enjoy your enhanced Discord server!
