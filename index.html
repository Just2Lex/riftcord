Command List

Comprehensive set of commands for moderation and League of Legends stats

Moderation

!ban @user Ban a user from the server
!kick @user Kick a user from the server
!mute @user Temporarily mute a user
!clear [number] Delete a specified number of messages

League of Legends

!stats [summoner] Get summoner statistics
!match [summoner] Show recent match history
!rank [summoner] Check current rank and LP
!live [summoner] Check if a summoner is in game

Utility

!help Show all available commands
!serverinfo Display server information
!userinfo [@user] Display user information
!ping Check bot response time

Setup Guide

1

Invite the Bot

Add RiftCord to your Discord server using the invite link. Make sure to grant necessary permissions.

Invite Now
2

Configure Settings

Use the !setup command to configure bot settings for your server and set up required channels.

3

Riot API Key (Optional)

For full LoL features, add your Riot API key using !setapikey [your_key] (admin only).

Get your API key from Riot Developer Portal

4

Customize & Enjoy

Customize moderation settings, enable/disable features, and enjoy your enhanced Discord server!
