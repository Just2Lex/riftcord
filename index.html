Installation Guide

1

Download BetterDiscord

Make sure you have BetterDiscord installed. You can get it from their official website.

2

Download the Theme

Download the latest version of RiftCord from the Releases page.

3

Install the Theme

Place the RiftCord.theme.css file in your BetterDiscord themes folder.

Usually located at: %appdata%/BetterDiscord/themes

4

Enable and Enjoy

Open Discord, go to Settings > BetterDiscord > Themes, and enable RiftCord.
